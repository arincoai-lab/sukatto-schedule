## アプリ名（20字）
```
スカッと予定：話す・撮るでカレンダー登録
```
> ブランド名「スカッと予定」= PWA manifest名と一致。OAuth同意画面の "Skatto Schedular" は開発者向け表記で別サーフェス＝凍結のまま影響なし。

## 短い説明（68字）
```
「明日15時 歯医者」と話すだけ。Google/Outlook/iCloudへ1タップ登録。端末内AIで解析、予定は預けません。
```

## 長い説明（約1,650字）
```
■ 予定入力、まだ手で打っていますか？

スカッと予定は「話す・撮る・打つ」だけで予定をカレンダーに登録できる、入力に特化したスケジュールアプリです。日時を一文字ずつ打ち込む手間をなくすために作りました。

「明日の15時から歯医者」と話す。紙の予定表を撮る。サッと一行打つ。あとは1タップ。それだけで予定が入ります。


■ 3つの入力方法

🎙️ 話す
「来週火曜の午後3時に歯医者」と話すだけ。音声認識とAIが日時とタイトルを自動で読み取ります。

📷 撮る
紙の月間予定表やホワイトボード、プリントを撮影。OCR＋AIが複数の予定をまとめて抽出します。

✏️ 打つ
一行入力、またはよく使う予定はテンプレを1タップ。「出勤」「ジム」も瞬時に登録できます。


■ 3つのカレンダーに、同時登録

仕事はOutlook、家族はGoogle、個人はiCloud——分かれていても大丈夫。選んだカレンダーすべてに一度で登録できます。Outlookなど公開URL（.ics）の外部カレンダーは閲覧にも対応。

・Google カレンダー（登録・閲覧・編集・削除）
・Outlook / Microsoft 365
・iCloud カレンダー


■ あなたの予定を、預かりません

写真とテキストの解析は、すべてあなたの端末の中で完結します。スカッと予定は予定の内容を保存するサーバーを持ちません。

・写真・テキストの解析は端末内のAIで完結
・カレンダーの認証情報は端末内のみ・私たちは受け取りません
・登録はあなたのカレンダーへ直接
・個人を識別しないプライバシー配慮型の計測のみ


■ こんな方に

・複数のカレンダーを使い分けていて、二重入力が面倒な方
・打ち込みが苦手・移動中や家事の合間に予定だけ入れたい方
・紙のプリントや予定表をデジタルにまとめたい方（子どもの学校行事など）
・自分の予定をむやみにサーバーに預けたくない方


■ 料金

無料で全部の入力方法を試せます。もっと使いたくなったら、買い切りのProで無制限に。月額はありません。

・無料：音声・写真・手入力すべて／3カレンダー同時登録／テンプレ5種・音声1日3回
・Pro（買い切り）：音声もテンプレも無制限。個人開発の応援にもなります。


■ 使い方

1. カレンダー（Google / Outlook / iCloud）を接続
2. 話す・撮る・打つで予定を入力
3. 内容を確認して1タップで登録

インストール不要のWeb版もあります：https://sukatto.rt-ai-lab.com


予定入力の手間を、ゼロに近づけよう。
```

## ASOメモ（検索で拾わせる設計）
- 主要KWを **タイトル ＞ 短い説明 ＞ 長い説明の冒頭** の順で厚く配置：`音声 / 予定 / カレンダー / 撮る(写真) / Google / Outlook / iCloud / 登録`。
- 狙う検索語：「カレンダー 音声入力」「予定 写真 読み取り」「google outlook 同時 カレンダー」「予定 管理 音声」。

## 公開時のチェック（コピー以外の運用）
- AAB署名：keystore `schedule/android-build/skatto-release.keystore`（Package ID `app.skatto.schedule`）。
- Play Console 登録（$25 一回）→ 内部テスト → 製品版。
- Data safety フォーム：端末内処理・予定内容は自社サーバー非保存・カレンダー権限の用途を申告（privacyポリシー `https://sukatto.rt-ai-lab.com/privacy` と整合）。
- 初期低評価回避：確実に動く端末から段階公開。
