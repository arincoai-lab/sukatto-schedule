# 統一ポジショニング

全チャネルでこの言葉を軸にし、表現をブラさない。

## コア3行
1. 話す・撮る・打つで、**予定入力は1タップ**。
2. **Google・Outlook・iCloud に同時登録**。分かれたカレンダーをまとめて。
3. **端末内AIで解析、予定は預けない**（プライバシー特化）。

## 媒体別ワンライナー
| 用途 | 文言 |
|---|---|
| ストア短い説明 / meta | 話す・撮る・打つだけで、Google・Outlook・iCloudに1タップ登録。 |
| 技術者向け（Zenn / X dev） | 端末内LLM×OCR×音声認識で「話すだけでカレンダー登録」を作った。予定はサーバーに送らない。 |
| 一般向け（note / SEO） | 「明日15時 歯医者」と話すだけ。予定入力のめんどくささをゼロに近づけるアプリ。 |

## 差別化の核（訴求の優先順）
1. **入力特化のラクさ**：話す/撮る/打つ → 1タップ。日時を手打ちしない。
2. **3強カレンダー同時登録**：Google / Outlook / iCloud を一度で。
3. **プライバシー**：解析は端末内で完結、予定内容を保存するサーバーを持たない。

## 想定ターゲット
- 複数カレンダーを使い分けて二重入力が面倒な人
- 打ち込みが苦手／移動中・家事の合間に予定だけ入れたい人
- 紙のプリント・予定表をデジタル化したい人（子の学校行事など）
- 予定をむやみにサーバーに預けたくない人

## 固定事項
- App名（OAuth同意画面）="Skatto Schedular" は凍結。日本語ユーザー向け表示名は「スカッと予定」（PWA manifest名）。
