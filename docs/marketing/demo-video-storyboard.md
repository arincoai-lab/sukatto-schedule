# デモ動画 / GIF 絵コンテ（最重要の使い回しアセット）

全チャネル（Play・Zenn・note・X・LP）で使う。15〜30秒。

## 絵コンテ
| 秒 | 画面 | 見せること |
|---|---|---|
| 0-3s | アプリ起動、🎙️話すボタン | 「めんどうな予定入力を…」テロップ |
| 3-8s | 「明日の15時から歯医者」と話す→波形 | 音声だけで入力している様子 |
| 8-13s | 自動でイベントカードに変換（日時・タイトル） | AIが解析する“気持ちよさ” |
| 13-18s | 接続中チップ（Google/Outlook/iCloud）→1タップ登録 | 3強に同時登録 |
| 18-23s | カレンダーに反映された画面 | 完了。「話すだけで、入る。」 |
| 23-27s | 🔒「解析は端末内・予定は預けません」 | 差別化の一撃 |

## 撮り方
- 実機のスカッと予定を画面収録（iOS/Android標準機能）。
- **BGMなし・字幕あり**（音声OFF環境でも伝わるように）。
- 実データで（架空でも自然な予定名で）。個人情報が映らないよう注意。

## 書き出し
- 動画本体（X / Play プロモ動画）。
- 先頭8秒をGIF化（記事・X埋め込み用）。
- **縦型（9:16）版**も1本あるとX/Playストアで有利。

## 使い回し先
- Google Play：プロモ動画（YouTube URL）＋スクショの元
- Zenn / note：記事冒頭にGIF
- X：初速投稿の主役
- LP：既にヒーローに実演カードあり（差し替え検討可）
